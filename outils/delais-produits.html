Délais produits — LeadTime DTW

Importez un fichier Excel contenant les délais historiques. L'outil calcule la moyenne par produit et génère un fichier DTW pour mettre à jour le LeadTime dans SAP (oItems).

1. Importer le fichier Excel

Colonne C = code produit, Colonne J = délai en jours. Le même code produit peut apparaître plusieurs fois — la moyenne sera calculée.
